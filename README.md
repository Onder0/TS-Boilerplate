# BOILERPLATE FULLSTACK NODEJS

## Installation

### With Git

```
git clone https://github.com/Onder0/boilerplate-F_JS
```

```
rm -rf .git
git init
```

### With pnpx

```
pnpx degit https://github.com/Onder0/boilerplate-F_JS
```

Install packages:

```bash
pnpm i
```

## Scripts

### Dev environment

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Prod

```bash
pnpm start
```

### Lint

```bash
pnpm lint
```
